# Docker 3-Tier Async Pipeline

A lightweight, full-stack sandbox designed to demonstrate container orchestration, isolated container networking, and the end-to-end request-response lifecycle across multiple decoupled microservices.

## 🏗️ Architecture Overview

This project layout showcases a strict separation of concerns, separating the front-end interface from internal core services using a middleware proxy/gateway.
